FreeWave is a decentralized music player built on Nostr.  
Users can send commands to play songs across devices in real-time.

## Features
- Play songs via Nostr commands
- Works on Termux (Android/Linux)
- Automatic download & play using `yt-dlp` and `mpv`
- Simple configuration
